# 未来功能存档

以下功能因需外部依赖暂未实现，保留设计待后续添加：

## PDF 工具（需外部服务）
- PDF → Word/Excel/PPT (需 LibreOffice 或商业库)
- PDF → Markdown (需解析引擎)
- PDF OCR 识别 (需 Tesseract)
- PDF 水印 (pdf-lib 可实现基础版，待完善)

## 图片处理（需 AI/外部库）
- 图片去背景 (需 AI 模型)
- 证件照换背景 (需 AI)
- 图片转文字 (需 OCR)

## AI 工具（需 API Key）
- 语音转文字 / 字幕生成 (需 Whisper)
- AI 生成 PPT / 简历 / 论文润色
- AI 图片修复/放大
- AI 文档摘要

## 音视频（需 ffmpeg）
- 音视频格式互转
- 视频裁剪/合并
- 音频提取
- 视频转 GIF
- 音频降噪

## 批量处理（部分可实现）
- 批量压缩（需 ffmpeg 支持视频）
- 批量格式转换（需 LibreOffice 支持文档）
